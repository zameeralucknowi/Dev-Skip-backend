# DevSkip API's

# AuthRoute
- POST /signup
- POST /login
- POST /logout

# ProfileRoute
- GET /profile
- PATCH /profile/edit
- PATCH /profile/password

# ConnectionRoute
- POST /request/dope/:userId
- POST /request/nope/:userId
- POST /request/review/accept/:requestId
- POST /request/review/reject/:requestId

# userRoute
- GET user/connections
- GET user/requests
- GET user/feed
